# PlayList Maker!
## слушай и создавай свои paly листы

Программа для поиска и прослушивания треков в iTunes. В ней вы можете составлять свои плай листы и сохранять в избранные треки

использованы технологии
- Kotlin
- MediaPlayer
- Room
- Koin
- Navgation
- Retrofit
- Glide
- Kotlin Coroutines
